# Battuta della Settimana 😂

Un sito semplicissimo che pubblica una battuta a settimana.

## Come funziona

1. Leggi l'indovinello
2. Clicca il bottone **"Scopri la battuta della settimana"**
3. Compare la risposta

Esempio:

**Domanda**
Che differenza c’è tra me e Batman?

**Risposta**
Nessuna, entrambi facciamo le bat-tute 🦇

## Tecnologie

Il sito è volutamente minimal:

* HTML
* CSS
* JavaScript

## Obiettivo del progetto

* Pubblicare una battuta a settimana
* Creare un piccolo progetto divertente
* Monetizzare con pubblicità online
